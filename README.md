# Credit Card Fraud Detection Project

## Project Overview

This project aims to detect fraudulent credit card transactions using Machine Learning techniques. The main goal is to classify transactions as either Fraudulent or Genuine based on transaction data.

Credit card fraud detection is one of the most important real-world applications of Data Science and Machine Learning in the banking and financial sector.

---

## Problem Statement

Credit card fraud causes huge financial losses every year. The challenge is to accurately detect fraudulent transactions while minimizing false alarms.

This project helps in identifying suspicious transactions using Logistic Regression classification.

---

## Dataset Used

Dataset Name: Credit Card Fraud Detection Dataset

Source: Kaggle

The dataset contains:

- Time
- Amount
- Transaction Features (V1 to V28)
- Class

Where:

- 0 = Genuine Transaction
- 1 = Fraudulent Transaction

## Project Workflow

### 1. Data Collection

Downloaded the dataset from Kaggle and stored it locally.

### 2. Data Preprocessing

- Loaded dataset using Pandas
- Checked missing values
- Verified data types
- Understood class imbalance

### 3. Exploratory Data Analysis

- Fraud vs Genuine transaction analysis
- Countplot visualization
- Data understanding

### 4. Feature Selection

Separated:

- Input Features (X)
- Target Variable (y)

### 5. Train-Test Split

Split data into:

- Training Data
- Testing Data

using train_test_split()

### 6. Model Building

Used Logistic Regression model for fraud detection.

### 7. Model Evaluation

Evaluated using:

- Accuracy Score
- Classification Report
- Confusion Matrix
